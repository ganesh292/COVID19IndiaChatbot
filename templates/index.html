COVID India Cases Live Tracker

Chat Interface to get realtime information on cases. Thanks to COVID19INDIA for the data source!

Welcome! To get started you could:

1. Enter State/District/City name to get the latest updates with case history

2. Type "Top 10 states" to get Top 10 affected States

3. Type "Top 10 districts of #State name(No spaces) " to get top 10 affected districts of a particular state

4. Type "Show states/districts/cities" to get the complete list
